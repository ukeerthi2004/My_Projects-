# Task Automation Script Suite (Python)

## Overview
A Python-based automation tool to perform routine operational tasks such as file sorting, backup automation, and report generation. The tool simulates real-world automation workflows by processing files from a configured source directory.

## Features
- Automated file sorting by file type
- Automated folder backup with timestamped copies
- Automated report generation with file statistics
- Modular function-based design
- Basic error handling for reliability

1. Place test files in the `sample_files/` folder.
2. Run the script:

3. Choose options from the menu to run automation tasks.

## Use Case
This project demonstrates Python-based task automation for file management, backup operations, and reporting, which are common in IT operations and backend automation workflows.
